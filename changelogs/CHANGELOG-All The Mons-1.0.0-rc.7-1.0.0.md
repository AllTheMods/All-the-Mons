# Changelog

# 📦 1.0.0

## 📰 General changes and notes

All the Mons leaves release-candidate status! 1.0.0 adds Legendary Monuments, a long awaited addition completing the collection of Legendaries available.
The pack is now considered "completable" with the final additions of recipes and quests to get you that Shiny Pika Star.


> ⚠️ **Summoning altar:** its input size was increased. Existing summoning altars must be **broken and placed again** to pick up the new config.
